# Common Requirements

This document outlines the common requirements for the project.

## Functional Requirements

1. **User personalization**: The application must allow users to select their preferred learning language and customize their learning paths based on their goals and proficiency levels. For web application a user have to download the built application and run it locally with local http server (maybe to add a bundle with http server and site). For mobile application user have to install the application from App Store or Google Play Store. 
1. **Main application Features**:
   - Interactive lessons
   - Progress tracking
   - Offline mode
   - Gamification
   - Speech recognition
   - Multimedia content
   - AI-powered recommendations
1. **Main leaning features**:
    - Vocabulary Builder
    - Grammar Exercises
    - Flashcards
    - Quizzes and Tests
    - Cultural Insights
1. **Main language functions**:
    - Vocabulary Management: The system must allow users to add, edit, and delete vocabulary words and phrases.
    - Grammar Management: The system must provide tools for users to practice and improve their grammar skills.
    - Pronunciation Practice: The application must include features for users to practice and improve their pronunciation.
    - Listening Comprehension: The system must offer listening exercises to help users improve their comprehension skills.
    - Reading Comprehension: The application must provide reading exercises to help users improve their comprehension skills.
